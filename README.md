# 🚢 Titanic Survival Prediction

## 📌 Overview

This project predicts whether a passenger survived the Titanic disaster using Logistic Regression.

---

## ⚙️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## 📊 Results

* Accuracy: ~75% – 82%
* Includes confusion matrix and classification report
